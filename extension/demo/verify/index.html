Mock 验证环境

用于在不访问真实 Git 平台 API 的情况下联调凭证验证 UI。

Mock Token 对照表

Token模拟结果
mock:success验证成功，用户 mock-user
mock:401HTTP 401: Bad credentials
mock:403HTTP 403: insufficient_scope
mock:timeout请求超时（约 12 秒）

使用步骤

  1. 打开扩展选项页，开启「使用 Mock 验证端点」
  2. 添加凭证，Token 填入上表中的 Mock Token
  3. 点击「验证凭证」，观察成功/失败 UI
  4. 侧边栏同步显示验证状态

说明

Mock 逻辑实现在扩展内 src/lib/verify/mock.js，本页面仅作开发文档与对照参考，不直接提供 HTTP 服务。
